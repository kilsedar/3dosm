Class: OSMBuildingLayer

OSMBuildingLayer(configuration, source)

Fetches OSM buildings, converts them to GeoJSON, and adds them to the WorldWindow.

Constructor

new OSMBuildingLayer(configuration, source)

Creates a sublass of the OSMLayer class.
Parameters:
Name Type Description
configuration Object Configuration is used to set the attributes of ShapeAttributes. Four more attributes can be defined, which are "extrude", "altitude", "altitudeMode" and "heatmap".
source Object Defines the data source of the layer. Its "type" can be either "boundingBox", "GeoJSONFile" or "GeoJSONData". If the "type" is "boundingBox", "coordinates" must be defined. The order of the "coordinates" is "x1, y1, x2, y2". If the "type" is "GeoJSONFile", "path" where the file resides must be defined. If the "type" is "GeoJSONData", "data" itself must be defined.
