Class: GeoJSONParserTriangulationOSM

GeoJSONParserTriangulationOSM(dataSource)

Triangulates polygons, which can be Polygons or MultiPolygons. Triangulated polygons improves rendering and painting performance compared to extruded polygons.

Methods

addRenderablesForMultiPolygon(layer, geometry, properties)

Invokes lateralSurfaces and/or topSurface to create a TriangleMesh for MultiPolygon geometry.

This method also invokes this GeoJSON's shapeConfigurationCallback for the geometry. shapeConfigurationCallback is extended by three attributes in the OSMBuildingLayer. These attributes are "extrude", altitude", "altitudeMode" and "heatmap". The altitude of the MultiPolygon is set using this function using setAltitude. If extrude and heatmap are enabled a new color is set for the MultiPolygon. If extrude is true, this function calls lateralSurfaces and topSurface. Otherwise it only calls topSurface.

Applications typically do not call this method directly. It is called by addRenderablesForGeometry.
