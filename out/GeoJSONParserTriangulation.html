Class: GeoJSONParserTriangulation

GeoJSONParserTriangulation(dataSource)

Triangulates polygons, which can be Polygons or MultiPolygons. Triangulated polygons improves rendering and painting performance compared to extruded polygons.

Methods

addRenderablesForMultiPolygon(layer, geometry, properties)

Invokes lateralSurfaces and/or topSurface to create a TriangleMesh for MultiPolygon geometry.

This method also invokes this GeoJSON's shapeConfigurationCallback for the geometry. It assumes extude and altitude is set in the configuration. If extrude is true, this function calls lateralSurfaces and topSurface. Otherwise it only calls topSurface.

Applications typically do not call this method directly. It is called by addRenderablesForGeometry.
Parameters:
Name Type Description
layer RenderableLayer The layer in which to place the newly created shapes.
geometry GeoJSONGeometryMultiPolygon The MultiPolygon geometry object.
properties Object The properties related to the MultiPolygon geometry.

lateralSurfaces(configuration, boundaries)

Creates a TriangleMesh for the lateral surfaces of polygons. It creates two triangles for each lateral surface.
Parameters:
Name Type Description
configuration Object Configuration is the object returned by shapeConfigurationCallback of GeoJSONParser or OSMBuildingLayer.
boundaries Object | Array.<Object> Boundaries of the polygons. If the geometry is Polygon the number of boundaries is one. If the geometry is MultiPolygon the number of boundaries is more than one.
